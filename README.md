# LegionScripts — DNA (Dawn of New Age)

Script Python per **[TazUO](https://github.com/bittiez/TazUO)** (client ClassicUO) da usare sullo shard custom italiano **[Dawn of New Age](https://www.dawnofanewage.it/)** di Ultima Online.  
Unisciti alla community su **[Discord](https://discord.gg/jkvsXsyDUZ)**.  
Vengono eseguiti dall'engine **LegionScripting** integrato in TazUO tramite IronPython — non c'è build step: basta avviare lo script in-game.

---


## Come usare gli script

1. Copia la cartella nel percorso `TazUO/TazUO/LegionScripts` (o una sottocartella al suo interno).
2. Apri TazUO e connettiti a **Dawn of New Age**.
3. Dal menu LegionScripting, seleziona lo script desiderato e avvialo.
4. Gli script a loop girano finche' non li fermi (o finche' non chiudi il gump / viene eseguito `API.Stop()`).
5. Per gli script one-shot l'esecuzione termina da sola.

> I file `.csproj` e `.sln` nella cartella padre esistono solo per il supporto IntelliSense — gli errori di build sono normali e irrilevanti.
